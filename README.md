# outman-cloud
 微服务学习相关代码示例
 
 # 一、子项目说明
    1、cloud-core 
    core核心包，用于存放公用的接口、bean对象、feign等。
    2、cloud-api
    用于提供对外接口的微服务，相当于消费者服务。
    3、cloud-service
    用于执行业务逻辑的微服务，相当于生产者服务。
    4、cloud-gateway
    统一网关出口，用于直接对外提供请求，并对请求进行初步处理，然后通过路由转发到相关内部服务。
    5、cloud-auth
    认证中心，用于控制微服务之间互相调用时，授权控制，防备非授权请求进入微服务组。
